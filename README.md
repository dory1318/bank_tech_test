# Bank Tech Test
based on the following requirements https://github.com/makersacademy/course/blob/master/individual_challenges/bank_tech_test.md

### To collaborate: 
``
git clone https://github.com/dory1318/bank_tech_test.git
``

``bundle install
``
### to run the test:
``
rspec
``
